# gwtransport1d

Compute the transport of temperature and contaminants in groundwater flow systems.

## Approach
1. The aquifer is characterized by estimating the aquifer pore volume distribution with heat as a tracer experiment. [./examples/01_Estimate_aquifer_pore_volume_from_temperature_response.py](https://github.com/bdestombe/python-gwtransport1d/blob/main/examples/01_Estimate_aquifer_pore_volume_from_temperature_response.py)
2. Using the aquifer pore volume distribution we compute:
  - The residence time [./examples/02_Estimate_the_residence_time_distribution.py](https://github.com/bdestombe/python-gwtransport1d/blob/main/examples/02_Estimate_the_residence_time_distribution.py)
  - Contaminent transport with different retardation factors
  - Log-removal of pathogens [./examples/03_Log_removal.py](https://github.com/bdestombe/python-gwtransport1d/blob/main/examples/03_Log_removal.py)

## Assumptions
- The aquifer pore volume distribution is constant over time.
- Currently, the Gamma-distribution is the only supported aquifer pore volume distribution

## Installation

```bash
pip install gwtransport1d
```
